Web Siteme AI Chatbot Nasıl 5 Dakikada Kurulur?

Tek satır kod veya eklenti paneli — STOAIX chatbot'u herhangi bir web sitesine teknik bilgi gerekmeden dakikalar içinde entegre edebilirsiniz.

Kısa yanıt: STOAIX chatbot kurulumu üç adımdan oluşur: hesap açın, script kodunu kopyalayın, sitenizin head bölümüne yapıştırın. WordPress, Wix ve Shopify için eklenti seçeneği de mevcuttur. Ortalama kurulum süresi 5 dakikadır, geliştirici gerekmez.

Hangi Platformlarda Çalışır?

STOAIX chatbot, HTML etiketi ekleyebildiğiniz her web platformuyla uyumludur:

WordPress
Wix
Shopify
Squarespace
Webflow
Custom HTML/CSS
React / Next.js
Vue / Nuxt

~30 sn

Script Nasıl Görünür?

Sitenize eklenen script örneği (placeholder — gerçek kod panelden oluşturulur):

<script
  src="https://cdn.stoaix.com/widget/v1/loader.js"
  data-key="YOUR_ACCOUNT_KEY"
  async
></script>

Bu tek satırı </body> etiketinin hemen öncesine yapıştırmanız yeterlidir. Geliştirici, FTP veya sunucu erişimi gerekmez.

WordPress İçin Eklenti Seçeneği

WordPress kullanıyorsanız script yapıştırmak yerine STOAIX'in WordPress eklentisini kullanabilirsiniz. Eklenti yönetim panelinden API key'inizi girin, kaydedip etkinleştirin — tamamdır.

Kurulum Site Hızını Etkiler mi?

STOAIX script'i async (asenkron) yüklenir; bu sayede sayfa yükleme sürenizi yavaşlatmaz. Widget kodu yalnızca kullanıcı sayfayı açtığında arka planda devreye girer.

Sıkça Sorulan Sorular

Chatbot kurulumu için geliştirici gerekiyor mu?

Hayır. STOAIX kurulumu tek bir script etiketi yapıştırmaktan ibarettir. WordPress, Wix veya Shopify kullanıyorsanız eklenti/app üzerinden geliştirici olmadan kurabilirsiniz.

Hangi platformlar destekleniyor?

HTML tabanlı tüm siteler, WordPress, Wix, Shopify, Squarespace ve özel geliştirme platformları desteklenmektedir. HTML tag eklenebildiği her site STOAIX ile uyumludur.

Kurulumdan sonra chatbot hemen çalışmaya başlar mı?

Evet. Script eklendikten ve hesap yapılandırması tamamlandıktan sonra chatbot anında aktif hale gelir. Test konuşması yaparak anında doğrulayabilirsiniz.

Chatbot'un görünümünü özelleştirebilir miyim?

Evet. STOAIX panelinden chatbot widget'ının rengini, konumunu, karşılama mesajını ve logo/marka kimliğini özelleştirebilirsiniz. Kodlama gerektirmez.

Kurulum site hızını etkiler mi?

STOAIX script'i asenkron yüklenir; yani sitenizin diğer içeriklerini yavaşlatmaz. Chatbot widget'ı sayfa yüklendikten sonra arka planda devreye girer.
