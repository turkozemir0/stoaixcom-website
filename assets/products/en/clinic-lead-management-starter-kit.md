# Clinic Lead Management Starter Kit

## Purpose

This kit helps your clinic create a simple but workable lead management system without jumping straight into a heavy CRM rollout.

## Core Pipeline Stages

1. `New Lead`
2. `First Contact Made`
3. `Waiting for Reply`
4. `In Follow-Up`
5. `Hot Lead`
6. `Appointment Booked`
7. `Lost / Archived`

## Daily Checklist

1. Have all new leads been reviewed?
2. Have missed calls been checked?
3. Are there any uncontacted leads left?
4. Have today’s follow-ups been completed?
5. Have near-term appointments been confirmed?

## Weekly Review

1. How many new leads came in?
2. How many became hot leads?
3. How many booked?
4. How many no-showed?
5. How many were lost?
6. Which stage leaked the most?
