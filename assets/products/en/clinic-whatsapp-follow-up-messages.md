# 25 Ready-to-Use WhatsApp Follow-Up Messages

## Usage Note

- Use the scripts as written or adapt them slightly by specialty.
- Do not stack messages too tightly together.
- End with a clear next step where possible.

## 1. First Follow-Up Messages

1. Hi, I wanted to follow up on your enquiry and see if I can help with the next step.
2. Hi, I’ve put together the key details for you. If useful, we can continue here.
3. Hi, I saw your message and wanted to make sure you got a proper reply from us.
4. Hi, if you’re still considering this, I’d be happy to walk you through it briefly.
5. Hi, I didn’t want your enquiry to get missed. Let me know if you’d like to continue.

## 2. For Price Enquiries That Went Quiet

6. Hi, I know price is usually only one part of the decision. If helpful, I can explain the options more clearly.
7. Hi, if anything about pricing or the process felt unclear, I’m happy to clarify it for you.
8. Hi, many patients pause at this stage because they still have one or two open questions. I can help with that.
9. Hi, if you’re still comparing options, I can give you a quick and clearer summary.
10. Hi, if you’d like, I can help you understand what would make the most sense for your situation.

## 3. For Hesitant Leads

11. Hi, taking time before deciding is completely normal. I can give you a short summary if that helps.
12. Hi, if you’re still unsure, we can go through the key points one by one.
13. Hi, there’s no rush. If there’s one thing you’re stuck on, I can help clarify it.
14. Hi, if useful, we can do a quick check-in and see what feels right for you.
15. Hi, whenever you’re ready to continue, I’m here.
